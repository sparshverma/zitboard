# ZitBoard Agency Execution Roadmap

This roadmap uses roles from The Agency to drive delivery from concept to launch.

## Phase 1: Positioning and IA (Week 1)

- Lead agents
  - product role agents from agency-agents/product/
  - UX architect: agency-agents/design/design-ux-architect.md
  - UX researcher: agency-agents/design/design-ux-researcher.md
- Outputs
  - ICP, JTBD, and top conversion goals
  - website information architecture and section blueprint
  - copy hierarchy: hero, proof, features, CTA

## Phase 2: Design System and Motion (Week 2)

- Lead agents
  - UI designer: agency-agents/design/design-ui-designer.md
  - visual storyteller: agency-agents/design/design-visual-storyteller.md
  - whimsy injector: agency-agents/design/design-whimsy-injector.md
- Outputs
  - color tokens (white, light teal, soft purple)
  - full-screen section design language
  - animation guidelines and reduced-motion strategy

## Phase 3: Frontend Build (Week 3-4)

- Lead agents
  - frontend developer: agency-agents/engineering/engineering-frontend-developer.md
  - software architect: agency-agents/engineering/engineering-software-architect.md
  - senior developer: agency-agents/engineering/engineering-senior-developer.md
- Outputs
  - responsive marketing website
  - animation-enabled hero and reveal-driven content sections
  - FAQ and pricing blocks based on provided SaaS overview

## Phase 4: Quality, Security, and Delivery (Week 5)

- Lead agents
  - code reviewer: agency-agents/engineering/engineering-code-reviewer.md
  - security engineer: agency-agents/engineering/engineering-security-engineer.md
  - devops automator: agency-agents/engineering/engineering-devops-automator.md
- Outputs
  - accessibility and performance quality gate
  - deployment checklist and monitoring hooks
  - launch readiness report

## Current Status

- In progress: Phase 3 initial marketing site implementation in ZitBoard.
- Next: Implement task management pages and analytics events, then finalize launch checklist.
